# Minimalist Line Art Assets - Ale & Pri Wedding

Premium single-line art illustration assets for the wedding landing page.

## Design Philosophy
- **Style**: Minimalist continuous line art on white background
- **Aesthetic**: Fine art, elegant, sophisticated
- **Technical**: No shading, no gradients, pure black lines (stroke #1A1A18)
- **Quality**: Clean lines, perfect geometry, high contrast

## Assets Included

### Characters
- `couple-silhouette.svg` - Elegant couple holding hands (200x200px)
- `toronto-lineart.svg` - Joyful dog in leaping pose (200x180px)

### Icons (Event Details)
- `calendar-lineart.svg` - Calendar with date (28x28px)
- `clock-lineart.svg` - Analog clock (28x28px)
- `location-lineart.svg` - Map location pin (60x60px)

### Section Icons
- `dressCode-lineart.svg` - Formal suit and gown (80x70px)
- `hotel-lineart.svg` - Hotel building (60x52px)
- `gift-lineart.svg` - Present with ribbon (60x56px)

## Stroke Settings
All SVGs use:
- **Stroke Width**: 2-2.5px (varies by size)
- **Stroke Color**: #1A1A18 (dark)
- **Fill**: None (transparent/white background)
- **Line Caps**: Round (smooth ends)
- **Line Joins**: Round (smooth corners)

## Usage
1. Embed directly in HTML: `<svg>...</svg>`
2. Reference as image: `<img src="icons/couple-silhouette.svg">`
3. Scale responsively with viewBox
4. Maintain aspect ratios for proper rendering

## Color Compatibility
- Works with any background color (pure black strokes)
- Originally designed for light background (#F0F8FF Sky Blue)
- High contrast ensures visibility on all backgrounds
